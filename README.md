KECNAS Bulut Depolama ve Resim Kaydetme Uygulaması

KECNAS, kullanıcıların resimlerini güvenli bir şekilde bulut ortamında depolamalarına olanak tanıyan bir uygulamadır.
Kullanıcılar resimlerini kolayca yükleyebilir, düzenleyebilir ve kaydedebilir.
PHP tabanlı bu uygulama, basit bir bulut depolama çözümü sunmaktadır.

Özellikler

Resim Yükleme: Kullanıcılar, görsellerini uygulamaya yükleyebilir.

Resim Depolama: Yüklenen resimler, güvenli bir şekilde bulut ortamında depolanır.

Dosya Yönetimi: Yüklenen dosyalar arasında düzenleme ve silme işlemleri yapılabilir.

Kullanıcı Yönetimi: Kullanıcılar, yükledikleri resimleri yönetebilir, listeleyebilir ve silebilir.

PHP ve MySQL: Uygulama, PHP ve MySQL kullanarak veritabanı bağlantısını sağlar ve resimlerin depolanmasını yönetir.

Teknolojiler

Backend: PHP

Veritabanı: MySQL

Frontend: HTML, CSS,

Dosya Depolama: Bulut depolama (dosyaların yerel sunucuda veya bulut ortamında saklanması sağlanabilir)

Gereksinimler

PHP 7.4 veya üzeri

MySQL (veritabanı yönetimi için)

Web Sunucusu (Apache veya Nginx)
